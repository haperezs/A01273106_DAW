Preguntas

  • ¿Qué es ODBC y para qué es útil?
    Open DataBase Connectivity. Es un estándar de acceso a las bases de datos para hacer posible el acceso a cualquier dato desde cualquier aplicación, sin importar qué sistema de gestión de bases de datos se utilice

  • ¿Qué es SQL Injection?
    Un tipo de ataque que aprovecha la vulnerabilidad de poder insertar codigo SQL en una base de datos

  • ¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?
    Limpiar las entradas, escapar los caracteres especiales, delimintar los valores de consultas, asignar minimos privilegios al usuario que conectará con la base de datos
