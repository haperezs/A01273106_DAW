Cuestionario

  • ¿Qué es entonces una base de datos - DB?
    Una base de datos es una colección de datos relacionados entre sí, estructurados y organizados

  • ¿En qué casos es conveniente usar bases de datos?
    Cuando se manejan grandes cantidades de información y es requerida por varios usuarios

  • ¿Qué es un sistema de gestión de base de datos - DBMS?
    Consiste en una colección de datos relacionados entre sí, estructurados y organizados, y un conjunto de herramientas para acceder a y gestionar esos datos

  • Enlista y explica las funciones/responsabilidades que tienen los DBMS.
    • Facilitar el acceso a los datos
    • Eliminar la inconsistencia en los datos
    • Controlar acceso concurrente y evitar anomalías derivadas
    • Brindar seguridad
    • Hacer copias de seguridad y restaurarlas

  • Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
    Para mi proyecto de diseño interactivo. El proyecto consistió en la creación de un sitio web llamado "TopTec" el cual tenía la intención de ayudar a estudiantes en varios aspectos relacionados con la vida estudiantil en el tec. El sitio presentaba lugares donde para comer, salir y frecuentar en general durante tu estancia en el tec. Toda esta información fue ingresada a mano por nosotros dentro del sitio. Una base de datos ubiera facilitado esto, permitiendo el acceso a usuarios y guardando la información de forma en que sea más dinámica de presentar
