Preguntas

  • ¿Por qué es importante hacer un session_unset() y luego un session_destroy()?
    session_unset() elimina las variables asociadas con la sesión y session_destroy() destruye la información registrada en una sesión. Por lo tanto se deber eliminar las variables primero o se podría volver a acceder a ellas.

  • ¿Cuál es la diferencia entre una variable de sesión y una cookie?
    Una variable de sesión guarda los valores del servidor. Una cookie guarda del lado del cliente la información del usuario

  • ¿Qué técnicas se utilizan en sitios como facebook para que el usuario no sobreescriba sus fotos en el sistema de archivos cuando sube una foto con el mismo nombre?
    Le asigna un nombre único con la fecha de subida

  • ¿Qué es CSRF y cómo puede prevenirse?
    Cross-site request forgery. Un tipo de ataque que explota la confianza que un sitio web tiene en un usuario particular, el atacante envía peticiones haciendose pasar por el usuario
