Preguntas

  • ¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications)?
    AJAX es una herramienta que facilita el desarrollo de RIA's por la posibilidad de cargar datos de manera asíncrona

  • ¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?
    Al igual que con JS, es vulnerable a diversos ataques como el XSS, por lo cual hay que realizar validaciones primero en el cliente

  • ¿Qué es JSON?
    JavaScript Object Notation. Un formato de texto que facilita el intercambio de datos entre el navegador y el servidor, notación literal de objetos de JavaScript
